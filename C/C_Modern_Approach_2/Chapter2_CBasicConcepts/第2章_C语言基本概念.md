# C 程序

某个人的常量可能是其他人的变量  

- 编译与链接  
  - 预处理 预处理器 preprocessor
  - 编译 编译器 compiler
  - 链接 链接器 linker
  
- 变量
  - 类型
  - 使用变量前需要对变量进行声明
  - 建议所有变量都进行初始化

- 常量
  - 使用宏定义 macro definition 给常量命名
  - 使用宏定义表达式时使用括号括起来

- 标识符
  - 下划线或字母开头
  - 标识符没有长度限制

- 关键字
  - keyword 对 C 编译器具有特殊含义不可作为标识符
  - 44 个关键字
